# Lottery Machine: Nixie 📟✨

A stylish and professional **Retro-Style Lottery Machine** built with **Flutter**. Featuring five glowing **Nixie tubes**, this app brings a vintage aesthetic to modern events, providing an exciting and visually captivating way to draw winners.

## 🌟 Key Features
- **Vintage Nixie Aesthetics:** Realistic animation and glowing effects of classic Nixie tubes.
- **Flexible Number Range:** Supports any range from **0 to 99,999**.
- **Smart Input Parsing:** Enter candidates using individual numbers (1,2,3), ranges (1-100), or mixed formats. Duplicates are automatically filtered.
- **Prize Registration:** Link specific numbers or ranges to prize names (e.g., `1:Space Travel`, `2-5:Smartphone`).
- **History Management:** Full control over the draw flow, including the ability to pre-set history.

## 💡 Best For...
- **Banquets & Parties:** Adding an exciting vibe to New Year’s parties or corporate events.
- **Festivals:** Professional-looking draws for neighborhood or school festivals.
- **Large Screen Displays:** Perfect for mirroring to a projector for a high-impact experience.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Advanced string parsing for range inputs and high-performance animation for Nixie tube lighting effects.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
